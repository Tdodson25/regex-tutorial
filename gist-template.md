# Tutorial: Regex matching a URL

Regex stands for "Regular Expression" It is used to replace objects in a string. 

## Summary

This can be used to match a URL

/^(https?:\/\/)?([\da-z\.-]+)\.([a-z\.]{2,6})([\/\w \.-]*)*\/?$/

This is asking if the input is a valid URL.
